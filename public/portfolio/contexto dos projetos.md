## **NoBuggy**

Site institucional desenvolvido para apresentar a NoBuggy como empresa de soluções digitais, com forte foco em performance, identidade visual e experiência do usuário.

O projeto foi construído com **React e Tailwind CSS**, utilizando **Framer Motion e GSAP** para animações avançadas e interações fluidas. Também foram implementados **scripts de envio automatizado de e-mails** a partir do formulário de contato e integração com **AWS S3** para armazenamento otimizado de vídeos e mídias, garantindo melhor desempenho e carregamento do site.

**Stack:** React, Tailwind CSS, Framer Motion, GSAP, AWS S3.

## **BeFest**

Plataforma completa de **agendamento e gestão de festas**, desenvolvida como um sistema full stack.

O projeto permite que o usuário **interaja com IA (GPT)** para planejar eventos, escolha pacotes personalizados e realize **pagamentos integrados via Mercado Pago**. Conta com **autenticação tradicional (login e senha)** e múltiplos níveis de acesso, incluindo **admin, fornecedor e usuário final**.

O sistema foi desenvolvido com **Next.js no front-end**, **Node.js no back-end** e **PostgreSQL** como banco de dados, garantindo escalabilidade e organização dos dados.

**Stack:** Next.js, React, Node.js, PostgreSQL, Mercado Pago, GPT API.

## **SofEmpire**

Sistema completo de **gestão de salões e profissionais da beleza**, desenvolvido sob demanda para um cliente do setor.

A plataforma permite controle total do negócio, incluindo **cadastro de serviços, matérias-primas, precificação automática**, cálculo de gastos fixos e variáveis, além de gestão financeira completa. ## **Meu prefeito (Aplicativo para Prefeituras)**

Aplicativo mobile desenvolvido para **prefeituras**, com foco em **transparência, comunicação pública e participação cidadã**.

A plataforma permite que cidadãos **registrem denúncias de problemas urbanos** (como iluminação pública, trânsito e segurança), acompanhem o **status das solicitações em tempo real**, recebam **avisos oficiais da gestão municipal** e acessem um **calendário de eventos públicos**. O sistema também conta com histórico de denúncias, visualização em mapa e interface simples e intuitiva para facilitar o uso por qualquer perfil de usuário.

O back-end foi construído de forma **serverless na AWS**, garantindo escalabilidade e baixo custo operacional.

**Stack:** React Native, AWS Lambda, AWS SES, DynamoDB, AWS S3.

## **Farmify**

Aplicativo mobile voltado para o **agronegócio**, criado para centralizar informações estratégicas e apoiar a tomada de decisão no dia a dia do produtor rural.

O app oferece **notícias do agro**, **cotações de commodities**, **cotações de moedas**, **previsão do tempo**, localização, modo claro/escuro e um **assistente de IA** capaz de responder dúvidas relacionadas ao setor. O sistema conta com **autenticação por login e senha** e foi projetado para ser escalável e facilmente extensível.

A arquitetura utiliza **múltiplos serviços**:

* Um **microserviço em Node.js** responsável pela sincronização de dados via scraping, utilizado para coletar informações que não estavam disponíveis em APIs públicas.
* Um **back-end em Python com Django Rest Framework**, que centraliza e disponibiliza os dados para o front-end.
* O banco de dados **PostgreSQL** é mantido sempre atualizado pelo serviço de sincronização, garantindo dados confiáveis e atualizados para o aplicativo.

**Stack:** React Native, Node.js, Python, Django Rest Framework, PostgreSQL, Web Scraping, Microserviços.
